Aurion Kodi Repository

Installation

To install this repository in Kodi:

  1. Download the repository addon: repository.aurion-1.0.0.zip
  2. In Kodi, go to Add-ons > Add-on browser (box icon)
  3. Select "Install from zip file"
  4. Navigate to and select the downloaded zip file
  5. After installation, you can install addons from "Install from repository > Aurion Repository"

Repository Contents

  • Repository addons list (addons.xml)
  • MD5 checksum (addons.xml.md5)
  • Browse addon packages (zips/)
